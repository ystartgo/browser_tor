# Secure Proxy Manager (安全代理管理器)

## English

**Secure Proxy Manager** is a Chrome extension designed to enhance your privacy by managing proxy settings and seamlessly integrating with the Tor Browser.

### Features

- **Direct Connection Mode**: Browse the web without any proxy.
- **Tor Proxy Mode**: One-click integration with Tor Browser (SOCKS5 `127.0.0.1:9150`).
- **Custom Proxy Configuration**: Configure your own SOCKS5, HTTP, or HTTPS proxy settings.
- **Tor Status Check**: Automatically detects if the Tor Browser is running and ready.
- **Easy Setup**: Provides instructions and a download link for Tor Browser if not detected.

### Installation

1.  Clone or download this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked** and select the directory containing this extension.

### Usage

1.  Click the extension icon in the toolbar.
2.  Choose **Direct** for normal browsing or **Tor Proxy** to route traffic through Tor.
3.  If using Tor Proxy, ensure Tor Browser is running. The status indicator will show green when connected.
4.  You can also configure custom proxy settings in the "Proxy Configuration" section.

## 繁體中文

**Secure Proxy Manager (安全代理管理器)** 是一個 Chrome 擴充功能，旨在透過管理代理設定並與 Tor 瀏覽器無縫整合，來增強您的隱私保護。

### 功能

- **直接連線模式**：無需代理即可瀏覽網頁。
- **Tor 代理模式**：一鍵整合 Tor 瀏覽器 (SOCKS5 `127.0.0.1:9150`)。
- **自訂代理設定**：設定您自己的 SOCKS5、HTTP 或 HTTPS 代理。
- **Tor 狀態檢查**：自動偵測 Tor 瀏覽器是否正在執行並準備就緒。
- **簡易設定**：若未偵測到 Tor 瀏覽器，提供說明與下載連結。

### 安裝方式

1.  複製 (Clone) 或下載此儲存庫。
2.  開啟 Chrome 並前往 `chrome://extensions/`。
3.  啟用右上角的 **開發人員模式 (Developer mode)**。
4.  點擊 **載入未封裝項目 (Load unpacked)** 並選擇包含此擴充功能的目錄。

### 使用說明

1.  點擊工具列中的擴充功能圖示。
2.  選擇 **Direct (直接)** 進行一般瀏覽，或選擇 **Tor Proxy** 透過 Tor 路由流量。
3.  若使用 Tor Proxy，請確保 Tor 瀏覽器正在執行。連線成功時，狀態指示燈將顯示為綠色。
4.  您也可以在「Proxy Configuration (代理設定)」區塊中設定自訂代理。
